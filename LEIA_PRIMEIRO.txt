## 📋 Se Algo Não Funcionar

### Backend não inicia
```
→ Verificar se MySQL está rodando
→ Verificar porta 8080 disponível
→ Ver logs do Spring Boot
```

### Frontend não carrega
```
→ Verificar npm está instalado (npm -v)
→ Limpar cache: npm cache clean --force
→ Deletar node_modules e refazer npm install
```

### Playground em branco
```
→ Abrir console (F12)
→ Ver se há erros
→ Tentar em navegador diferente
→ Limpar cookies
```

### API não responde
```
→ Verificar Backend está rodando
→ Verificar NEXT_PUBLIC_API_BASE_URL=http://localhost:8080
→ Verificar porta 8080 no firewall
```
